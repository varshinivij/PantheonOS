## Installation

```bash
git clone https://github.com/aristoteleo/pantheon-agents.git
cd pantheon-agents
pip install -e ".[toolsets]"
```

## Pantheon REPL

The easiest way to start using Pantheon is through the CLI interface:

```bash
# Start with default settings
python -m pantheon.repl
```
