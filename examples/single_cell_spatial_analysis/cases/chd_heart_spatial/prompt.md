# Integrative analysis of human heart spatial transcriptomics data and human fetal heart development single cell transcriptomics data

Main datasets:
- Human heart spatial transcriptomics data(2D): 
  + MERFISH
  + Timepoint: pcw 12
  + Paper: https://www.nature.com/articles/s41586-024-07171-z
  + ./merfish_human_heart_2d.h5ad
- Human heart spatial transcriptomics data(3D): 
  + MERFISH
  + Timepoint: pcw 12
  + Paper: https://www.biorxiv.org/content/10.1101/2025.11.02.686137v1
  + ./merfish_human_heart_3d.h5ad
- Human fetal heart development single cell transcriptomics data:
  + 10x Genomics Multiome (scRNA-seq part)
  + Paper: https://www.medrxiv.org/content/10.1101/2024.11.20.24317557v2
  + ./human_fetal_heart.h5ad

Other datasets:
- CHD(congenital heart disease) genes list
  + source: https://chdgene.victorchang.edu.au/
  + ./chd_genes.tsv


Based on the above datasets, please firstly read the datasets and understand the structure of the datasets.
Then design a comprehensive analysis plan for the integration of those datasets,
try to figure out some interesting questions and answers to the questions by analyzing the datasets.
Then perform the analysis and report the results through a markdown file(including the results explanation and the figures).
